# 更新日志

本项目所有功能迭代记录，按时间倒序排列。

## [未发布]

### 新增
- 记录列表显示照片缩略图条（最多 4 张，超出显示 +N）

- 政府往来、维权投诉类型详情页不再显示关联记录模块

## v0.4.0 — 详情页 UI 优化

### 新增
- 记录详情页照片与附件拆分为两张独立卡片，分别显示文件数量
- 照片网格改为 `grid-cols-3 / sm:grid-cols-4`，正方形裁切展示，支持灯箱放大
- 上传入口由大型虚线框改为 slot 标题旁的小型「+ 上传」按钮，支持拖拽整个区域
- Slot 标题行新增下划线分隔线与图片数量徽章
- 解除关联记录改为遮罩确认弹窗（原 `window.confirm`）
- 记录列表删除改为遮罩确认弹窗

### 变更
- 记录详情页布局重整：类型徽章 + 彩色日期标签置顶，参与人员 chip 紧跟标题，描述无标签直接展示

## v0.3.0 — 权限与账号系统

### 新增
- JWT 登录认证（`jose` + `bcryptjs`），HttpOnly Cookie 存储，30 天有效期
- 中间件全局路由保护，未登录自动跳转 `/login`
- 管理员 / 普通成员双角色体系
- 导航栏用户菜单（用户名、管理员徽章、退出按钮）
- 设置页新增「账号管理」Tab（管理员专属）：新建账号、重置密码、切换角色、删除账号
- 全部写操作 API 路由添加 `requireAdmin()` 鉴权守卫（14 处）
- 非管理员隐藏所有新增/编辑/删除入口（布局、列表、详情、设置页均处理）
- 初始管理员账号自动创建（默认 `admin / admin123`，可通过环境变量 `INITIAL_USERNAME` / `INITIAL_PASSWORD` 覆盖）

## v0.2.0 — 合同管理 & 往来单位

### 新增
- 合同管理模块：新建、编辑、删除合同，支持维保/管理/维护/工程四种类型，收入/支出方向
- 合同附件上传与预览
- 合同统计面板：总数、收入合计、支出合计、即将到期数量（可点击筛选）
- 到期状态自动标记（已到期 / 即将到期 30 天内）
- 往来单位管理：设置页维护，新建合同时可 Combobox 选择，保存时自动入库
- 首页新增「即将到期合同」提醒卡片

---

## v0.1.0 — 核心记录系统

### 新增
- 工作记录 CRUD：会议记录、公告通知、政府往来、维权投诉四种类型
- 记录列表分页、关键词搜索、类型筛选、日期范围筛选
- 记录详情页展示参与人员（成员 chip + 自由文本）
- 附件分类上传：照片槽位（现场照片/记录照片/全景照片/近景照片）+ 文档槽位
- 附件图片在线预览（灯箱）、文档下载
- 关联记录模块：会议 ↔ 公告互相关联/派生，支持搜索已有记录后关联
- 成员管理：设置页维护业委会委员名单，记录表单多选参与人
- 导航栏左上角 Logo、深色主题，记录类型彩色边框与徽章
- SQLite 数据库（`better-sqlite3`），数据目录与上传目录自动创建
- 附件文件以 UUID 重命名存储，通过 `/api/attachments/[id]/download` 提供下载
